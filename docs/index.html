⌬ subnetting-sim
Código Seguridad

Simulador de subnetting 100% en tu navegador

Calcula redes IPv4/IPv6, divide en subredes, haz VLSM, resume supernets y verifica IPs. La lógica es el mismo Python del repo ejecutado en el navegador con Pyodide (CPython en WASM): sin servidor, sin enviar tus datos a ningún sitio.

Inicializando entorno Python (Pyodide)… primera carga ~10 s

Si rellenas ambos se usa "count".

Atajos

  • info 192.168.1.10/24 — análisis completo
  • split 10.0.0.0/24 --hosts 50 — subredes de 50 hosts
  • vlsm 192.168.1.0/24 --needs 50,25,10,2
  • supernet 192.168.0.0/24 192.168.1.0/24
  • verify 192.168.1.50 192.168.1.0/24 — la máscara es obligatoria en IPv4

💡 En IPv4 la máscara es obligatoria: usa 192.168.1.0/24 o 192.168.1.0 255.255.255.0. IPv6 sin prefijo se trata como /128.

¿Prefieres la terminal? Hay una CLI: subnetcalc info 192.168.1.10/24.
